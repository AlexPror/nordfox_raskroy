# NordFox Raskroy

Приложение для линейного раскроя профилей по Excel-спецификации с акцентом на профильные коды NordFox (серии Н20-Н23), поддержкой склада обрезков и экспортом результатов.

## Обновление 0.4.0

- **Габариты сечения:** модуль `profile_dimensions.py` (Н20–Н24, L15/L16/L20, DT20/DT21) для расчётов и справки.
- **Рез с обрезка:** при выборе обрезка учитывается запас по правому торцу под угловой рез (угол правого подреза + max сторона сечения); при нехватке длины деталь уходит на новый пруток.
- **Схема / альбом (Qt, PDF):** техотступ и пропил выделены цветом, пропил вертикальный; мм теха над полосой; без линий углов на профиле; PDF-альбом с кириллицей (подключение Arial/DejaVu через ReportLab).
- **Тесты:** `pytest`, опция установки `pip install -e ".[test]"`, workflow GitHub Actions `.github/workflows/pytest.yml`.

## Обновление 0.3.0

- Схема раскроя: подписи углов `L/R` на деталях и отдельная легенда `#N` для коротких сегментов.
- PDF схемы: горизонтальная (landscape) компоновка, легенда коротких сегментов с углами и источником.
- Массовый баланс: контроль, что масса обрезков не превышает массу закупки.
- Навигация: zoom только по `Ctrl + колесо`, обычное колесо снова прокручивает.
- Обрезки: переиспользование по совместимому ключу профиля для корректного дораскроя.

## Что умеет

- Считывает спецификацию `.xlsx` и разворачивает количество деталей в набор требований.
- Фильтрует профили по сериям `0..3`:
  - `0 = Н20`
  - `1 = Н21`
  - `2 = Н22`
  - `3 = Н23`
- Оптимизирует раскрой по выбранным длинам заготовок (`6000 / 7500 / 12000` мм).
- Учитывает пропил (`kerf`, в длине съёма **2×kerf** на деталь) и минимальный остаток.
- Техотступ по **первому** углу детали (90° → 30 мм, иначе 50 мм); второй угол — в подписи и в геозапасе правого торца при обрезке.
- Может использовать начальный склад обрезков из отдельного Excel.
- Показывает результат в таблице с редактированием и повторным пересчётом.
- Считает массу каждой строки (`Масса профиля, кг`) и итог массы внизу таблицы.
- Экспортирует результат в Excel и PDF с цветами по модулям.
- Включает режим подбора наилучшего набора длин заготовок.

## Технологии

- Python 3.10+
- `PySide6` (основной GUI)
- `openpyxl` (Excel)
- `reportlab` (PDF)
- `pytest` / `unittest` (тесты)

## Установка

```bash
python -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
```

Альтернатива через пакет:

```bash
pip install -e .
```

## Быстрый старт

Запуск Qt-приложения:

```bash
set PYTHONPATH=src
python -m nordfox_raskroy
```

или:

```bash
run.bat
```

Если `PySide6` не установлен, можно запустить fallback на tkinter:

```bash
python -m nordfox_raskroy.app_tk
```

## Входные данные

### 1) Спецификация

Основной файл `.xlsx` с деталями. Ключевые поля в строке:

- модуль
- тип профиля (например `СК-1-456`, `СС-2-554`, `Р-0-1230`)
- длина
- угол
- количество

### 2) Склад обрезков (опционально)

Отдельный `.xlsx`:

- колонки: `Длина`, `Количество`
- лист: `Склад` (или активный лист)

Генератор примера:

```bash
python scripts/generate_scrap_stock_example.py
```

## Материалы и масса

Справочник `кг/м` вынесен в единый модуль:

- `src/nordfox_raskroy/materials_library.py`
- документированная таблица: `docs/MATERIALS_TABLE.md`

Что покрыто:

- серии NordFox `0..3` (Н20-Н23)
- дополнительные профили (`DT`, `L15`)
- типовой стандартный металлопрокат по подстрокам в наименовании

В таблице результата:

- отдельная колонка `Масса профиля, кг`
- строка `Итого масса профиля, кг` внизу

При пересчёте по таблице колонка массы используется только как отображение и не влияет на алгоритм оптимизации.

## Интерфейс Qt

Основные функции:

- выбор файла спецификации и файла склада обрезков
- фильтр профилей Н20-Н23 (цифры `0..3`)
- выбор длин заготовок
- настройка `kerf` и минимального остатка
- расчёт и сортировка результата
- редактирование строк результата с повторным пересчётом
- экспорт в Excel/PDF
- подбор рекомендованного набора длин заготовок

## Экспорт

### Excel

- лист `Раскрой`
- лист `Цвета модулей`
- (опционально) лист `Сводка`
- включает колонку `Масса профиля, кг` и итоговую строку

### PDF

- таблица раскроя с цветами строк
- колонка `Масса профиля, кг`
- итог массы в конце таблицы

## Структура проекта

```text
docs/                         Документация (включая таблицу материалов)
scripts/                      Генераторы тестовых входных файлов
src/nordfox_raskroy/
  app.py                      Точка входа GUI
  qt_app.py                   Основное Qt-приложение
  app_tk.py                   Fallback GUI на tkinter
  excel_io.py                 Разбор спецификации Excel
  optimizer.py                Алгоритм раскроя
  materials_library.py        Библиотека kg/m и расчёт массы
  export_results.py           Экспорт в Excel/PDF
  table_demand_import.py      Импорт отредактированной таблицы
  profile_codes.py            Разбор кодов СК/СС/Р и серии
  profile_dimensions.py       Габариты сечения профилей (мм)
test/                         Тесты
```

## Тестирование

Рекомендуется установить зависимости для тестов и запускать `pytest` (см. `pyproject.toml`, секция `[project.optional-dependencies] test`):

```bash
pip install -e ".[test]"
pytest
```

Либо из `requirements.txt` (включает `pytest`):

```bash
pip install -r requirements.txt
set PYTHONPATH=src
python -m pytest test -v
```

Полный прогон через unittest (без pytest):

```bash
set PYTHONPATH=src
python -m unittest discover -s test -p "test_*.py" -v
```

## Версионирование

Версия хранится в `src/nordfox_raskroy/VERSION` и подтягивается в метаданные пакета при сборке/установке.

Проверка версии:

```bash
python -m nordfox_raskroy --version
```

## Типовые сценарии ошибок

- Нет данных после фильтра профилей -> проверить галочки серий `0..3` и формат кода профиля.
- Не читается файл Excel -> проверить, что файл существует и не открыт в режиме блокировки.
- Нет PDF-экспорта -> установить `reportlab`.
