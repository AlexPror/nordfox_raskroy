# План развития NordFox — раскрой

Текущая версия: см. `src/nordfox_raskroy/VERSION`.

## Уже совместимо (без доработки логики)

Обозначения **без** суффиксов В/Н/Л/П, только длина в коде, например:

- `СК-1-456`, `СС-2-554`, `Р-0-1230`

Разбор серии Н20–Н23 использует шаблон начала строки `СК|СС|Р` → цифра `0…3` → дефис; **хвост** (`456`, `1045Л` и т.д.) на классификацию не влияет. Эти строки должны попадать в расчёт при включённых соответствующих галочках профилей.

**Риск:** регрессии при изменении regex или фильтра — зафиксировать **юнит-тестами** на таких строках (см. фазу 0).
