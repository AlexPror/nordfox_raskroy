# Справочник массы погонного метра (кг/м)

Источник в коде: `src/nordfox_raskroy/materials_library.py`  
Профили **Н20–Н23** и **DT / L15** — из проекта `nordfox_specification` (`materials_database.PROFILES_MASS_PER_METER`).  
**Металлопрокат** — типовые ориентиры (ГОСТ/каталоги); при закупке уточнять по сертификату.

---

## 1. Каркас NordFox: цифра в коде `СК-` / `СС-` / `Р-` (0…3)

| Цифра | Серия | кг/м | Примечание |
|------:|-------|-----:|------------|
| 0 | Н20 | 1.0315 | H20 |
| 1 | Н21 | 1.206 | H21 |
| 2 | Н22 | 1.2075 | H22 |
| 3 | Н23 | 1.209 | H23 |

Коды без буквенных флагов в конце (`СК-1-456`, `Р-0-1230`) учитываются так же: важны префикс и **первая** цифра после него.

---

## API

- `kg_per_meter_from_profile_code(code)` — для строк спецификации в формате NordFox.
- `kg_per_meter_from_description(text)` — для произвольного текста (швеллер, труба и т.д.).
- `total_mass_kg(length_mm, kg_per_m, quantity)` — масса отрезка.
